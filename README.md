# crown-clothing

## What this App does?

- By this app, you will be able to:

  - Add or remove items from cart
  - Checkout with Stripe.

## Built with

- React
- Redux
- Firebase
- Styled-components

## Setup

- Clone the repository
- Use `cd <repository-path>`
- Run `yarn` or `npm i`
- Run `yarn dev` or `npm run dev`
